
# Prompt Hub (Static Site)

A minimal, fast, copy-to-clipboard Prompt Hub generated from a Notion export.

## How to publish on GitHub Pages

1. Create a new public repository on GitHub, e.g. `prompt-hub`.
2. Upload the files from this folder: `index.html`, `style.css`, `script.js`, `prompts.json`.
3. In the repository, go to **Settings â†’ Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` (or `master`) and folder **/ (root)**. Save.
6. Wait until GitHub Pages shows the site URL. Done.

## Local development
Just open `index.html` in your browser. No build step needed.
